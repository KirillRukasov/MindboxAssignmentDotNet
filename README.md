# Библиотека расчета площади

## Постановка задачи
Задача заключается в создании библиотеки для C#, которая будет рассчитывать площадь круга по радиусу и треугольника по трем сторонам. Так же должны присутствовать тесты для проверки корректности работы библиотеки, легкость добавления новых фигур и расчетов площади, и расчет площади фигуры без знания ее типа во время компиляции.

## Что сделано дополнительно
Создал веб приложение, которое предоставляет интерфейс взаимодействия пользователя с написанной библиотекой. Был использован React для фронтенда и Web API на ASP.NET Core для взаимодействия с бэкендом. Пример взаимодействия приведен в видеофайле ReactAppVideo.mp4

## Описание решения
Решение задачи включает в себя классы для создания экземпляров треугольника (`Triangle`) и круга (`Circle`), оба класса наследуются от интерфейса `IShape`. Основным классом является `AreaCalculation`, который предоставляет доступ к методу `CalculateShapeArea` для вычисления площади. Наследование общего интерфейса `IShape` позволяет добавлять новые фигуры и расчеты площади без изменения интерфейса клиента.

Решение также включает в себя тесты для проверки корректности работы библиотеки и возможность определения прямоугольности треугольника. 

## Запуск решения
Для запуска решения нужно создать экземпляр класса `AreaCalculation` и вызвать метод `CalculateShapeArea` с параметрами реализующие интерфейс `IShape` в качестве аргумента. В текущем решении это 'Triangle' и 'Circle'.
'''
var service = new AreaCalculation();
var circleArea = service.GetArea(new Circle(5));
'''

Другой вариант заключается в прямом вызове метода `CalculateArea()` на объекте, который реализует интерфейс `IShape`.
'''
var triangle = new Triangle(sideA, sideB, sideC);
var actualArea = triangle.CalculateArea();
'''

Для проверки треугольника на прямоугольность используется 'IsRightTriangle' метод в классе 'Triangle'.
'''
var result = triangle.IsRightTriangle();
''''

## Тестирование
Для тестирования используется 'xunit'. Для запуска тестов нужно открыть решение в Visual Studio и запустить тесты через меню Test -> All Tests. Или использовать команду [dotnet test](https://learn.microsoft.com/en-us/dotnet/core/tools/dotnet-test) в терминале. Все тесты содержатся в сборке 'AreaCalculatorDotNetLibrary.Tests'.

## Заключение
Библиотека предоставляет простую и гибкую библиотеку для вычисления площади круга и треугольника. Библиотека спроектирована таким образом, чтобы облегчить добавление новых фигур и расчет их площади без изменения API клиента.

Все решение можно разбить на два слоя: бэкенд и фронтенд. В идеале нужно разделить на два разных проекта. В данном случае, для удобства презентации результата, два слоя были объединены в один проект.

### Бэкенд на ASP.NET Core Web API
Бэкенд реализован с использованием технологии ASP.NET Core Web API и предоставляет API-методы для расчета площади геометрических фигур. В качестве основы для расчета площади используется библиотека AreaCalculatorDotNetLibrary, содержащая класс AreaCalculator и интерфейс IShape.

### Фронтенд на React
Фронтенд приложения разработан на React и предоставляет пользовательский интерфейс для ввода параметров геометрических фигур и отображения результатов расчета площади. Фронтенд осуществляет обмен данными с бэкендом через API-методы, предоставляемые сервером на ASP.NET Core Web API.
